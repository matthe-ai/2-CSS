CSS interno

Ao invés de fazer o style dentro da propria tag a ser personalizada, é atribuido uma classe a esta tag e a personalização será feita em um espaço a parte criando uma tag style dentro da tag head.

Dentro dessa tag style você especifica que item você quer mexer e abre chaves "{}" que é onde as personalizações irão ficar.



CSS externo

O CSS externo é quando as personalizações ficam dentro de um arquivo CSS externo que vai ser referenciado dentro das paginas html

Propriedade color

A propriedadecolor é a cor do texto, ou seja, você define a cor do seu texto utilizando a propriedadecolor. Essa cor pode ser definida ao colocar:

  • Nome da cor;
  • Código hexadecimal da cor;
  • Código RGB da cor.

Propriedade background

A propriedade background serve para definir um fundo para algum espaço, esse fundo pode ser uma imagem estatica, uma imagem e etc.


Com background

Sem background

Mas o background pode colocar tanto uma cor utilizando: background-color quanto pode colocar uma imagem no fundo utilizando: background-image


Tag Função
background-color Adicionar uma cor de fundo
background-image Adicionar uma imagem de fundo (seguido do url que é a localização da img)

Propriedades do background-image

  • background-size: contain & cover
    • contain: significa que a imagem vai se corter e não se esticar para preencher todo o espaço
    • cover: significa que a imagem vai cobrir todo o espaço
  • background-repeat: repeat & no-repeat
    • repeat: se a imagem for menor que o espaço, ela vai se repetir para cobrir ele por completo
    • no-repeat: se a imagem for menor que o espaço, ela não vai se repetir para cobrir ele

Propriedade border

A propriedade border serve para dar uma borda a "caixa" que será definida de forma similar a colocar borda no Excel, você pode definir a grossura da borda, o tipo dela e a cor dela.


Sem borda.


Com borda.


Sobre os tipos, existem estes:

tag função
solid Faz uma linha reta
dotted Faz uma linha de pontos
dashed Faz uma linha tracejada
double Faz duas linhas retas
groove Dá um certo efeito de 3d, profundidade

Além desses atributos, dá para especificar onde a borda pode ser colocada ao adicionar na tag border a direção, sendo:

  • top: topo
  • right: direita
  • left: esquerda
  • bottom: em baixo

Assim como também dá para deixar ela arredondada nas pontas bastando utilizar a tag border-radius e colocar uma medida em px ou porcentagem.
