CSS interno

Ao invés de fazer o style dentro da propria tag a ser personalizada, é atribuido uma classe a esta tag e a personalização será feita em um espaço a parte criando uma tag style dentro da tag head.

Dentro dessa tag style você especifica que item você quer mexer e abre chaves "{}" que é onde as personalizações irão ficar.



CSS externo

O CSS externo é quando as personalizações ficam dentro de um arquivo CSS externo que vai ser referenciado dentro das paginas html

Propriedade color

A propriedadecolor é a cor do texto, ou seja, você define a cor do seu texto utilizando a propriedadecolor. Essa cor pode ser definida ao colocar:

  • Nome da cor;
  • Código hexadecimal da cor;
  • Código RGB da cor.

Propriedade background

A propriedade background serve para definir um fundo para algum espaço, esse fundo pode ser uma imagem estatica, uma imagem e etc.


Com background

Sem background

Mas o background pode colocar tanto uma cor utilizando: background-color quanto pode colocar uma imagem no fundo utilizando: background-image


Tag Função
background-color Adicionar uma cor de fundo
background-image Adicionar uma imagem de fundo
